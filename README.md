## Installation

Install nft-market ReactProject with yarn

```bash
  cd nft-market
  yarn install
  yarn start
```
    

## Deployment

To deploy this project run

```bash
  cd nft-market
  yarn build
  then zip the build folder then upload to your host
  unzip build.zip then use it
```
